Project Name: DigiTools Platform

Description:
A modern web application where users can browse digital tools, view product details and add them to a cart. The app includes dynamic UI toggling between products and cart, real-time cart updates, and a smooth user experience with notifications and responsive design.

Technology used:
  1.React.js
  2.Tailwind CSS + DaisyUI
  3.JavaScript (ES6+)
  4.React-Toastify (NPM Package)
  5.JSON Data

3 Key Features:
  1.Add / Remove products from cart
  2.Toggle between Products & Cart view
  3.Toast notifications for all actions
